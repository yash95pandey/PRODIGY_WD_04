# My Portfolio 

This is a personal Portfolio Website built using HTML, CSS, and JavaScript. It showcases the work and skills of Yash Pandey, a web developer with a passion for IoT and tech innovation. The website features several sections including a brief introduction, an about me page, a collection of projects, and contact information.

# Features:

Header Section: Includes the portfolio owner's name, a brief tagline, and a circular profile image.

Navigation: Easy-to-use navigation links for smooth scrolling between sections (Home, About, Projects, and Contact).

About Me Section: Highlights Yash's education, skills, and experience in web development and IoT.

Projects Section: Displays a collection of notable projects like "IoT Home Automation," "Stopwatch Web App," and "Tic Tac Toe Game," with brief descriptions.

Contact Information: Provides easy access to Yash’s email, LinkedIn, and GitHub profiles.

Responsive Design: The layout adapts to different screen sizes, making it accessible on both desktop and mobile devices.

# Technologies Used:

HTML: Structure and layout of the portfolio site.

CSS: Styling for the site, including responsive grid layout and visual effects.

JavaScript: Adds interactivity to elements, such as smooth scrolling navigation.

# How It Works:

The page is divided into different sections, each with its own content, such as a welcome message, about information, and project details.

Clicking on the navigation links scrolls the user to the corresponding section.

The "Projects" section displays a grid of the portfolio’s projects with descriptions, and each project showcases different aspects of Yash’s development skills.

# Installation:

To use this portfolio, simply clone this repository to your local machine:

  git clone https://github.com/Yash/Portfolio.git


Open the index.html file in your browser, and you’ll have access to the full portfolio.
